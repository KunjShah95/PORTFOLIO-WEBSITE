 🚀 Kunj Shah - Portfolio Website  
Welcome to the repository of **Shah Kunj's Portfolio Website!  
This is a fully responsive and visually appealing portfolio website designed for showcasing skills, projects, and achievements. Built with modern web technologies, it incorporates interactive elements, animations, and a sleek UI/UX to create an outstanding personal brand presence.


 🌟 Features

🌐 Modern UI/UX Design
- Dark mode with neon accents.
- Full-screen, scroll-based sections for immersive navigation.
- Smooth transitions and hover effects.

💻 Technologies Used
- HTML5: Structuring the website.
- CSS3: Styling with advanced animations and responsiveness.
- JavaScript: Adding interactivity and dynamic content.
- Three.js: For 3D interactive visuals.
- GSAP: Smooth animations and scrolling effects.

🎨 Visual Highlights
- 3D rotating cube in the hero section using Three.js.
- Skill-based animated counters and hover effects.
- Responsive grid layout for projects and skills.



 📂 Folder Structure

PORTFOLIO-WEBSITE/
├── index.html          # Main HTML file
├── styles.css          # Styling for the website
├── script.js           # JavaScript for interactivity and animations
├── images/             # Folder containing all images and icons
├── libs/               # External libraries (Three.js, GSAP)
└── README.md           # Documentation file

 🛠️ Getting Started
Follow these steps to set up the project locally:

1. Clone the Repository
bash
git clone https://github.com/KunjShah95/PORTFOLIO-WEBSITE.git
cd PORTFOLIO-WEBSITE


2. Open the Project
- Open `index.html` in your favorite browser.

3. Customize
Feel free to modify the content in `index.html` and styles in `styles.css` to make it your own.
